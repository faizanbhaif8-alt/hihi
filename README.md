# Hihi Bot

A simple web-based bot that responds to the word 'hihi'.

## Features
- Responds with a friendly message when 'hihi' is typed.
- Simple and intuitive user interface.

## Setup Instructions
1. Clone the repository.
2. Open `index.html` in your web browser.
3. Type 'hihi' in the input box and click 'Send' to interact with the bot.
